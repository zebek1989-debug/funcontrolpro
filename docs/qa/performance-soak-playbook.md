# Performance And Soak Playbook (Milestone 5.2)

Last updated: 2026-03-15

## Objective

Standardize how we execute:
- 24h monitoring soak test,
- control stress scenarios,
- baseline capture for CPU/RAM/startup,
- crash diagnostics review.

## Inputs

- Release build from current main branch.
- Hardware target present in `docs/qa/hardware-matrix.csv`.
- `supported-hardware.md` aligned with tested branch.

## Main Artifacts

- `docs/qa/performance-baselines.csv`
- `docs/qa/performance-run-template.md`
- `artifacts/perf/<timestamp>/` (generated by soak script)

## Soak Run (24h)

Recommended command on Windows:

```powershell
pwsh -NoProfile -File scripts/performance/Run-Phase52Soak.ps1 `
  -ProjectPath src/FanControlPro.Presentation/FanControlPro.Presentation.csproj `
  -DurationHours 24 `
  -SampleIntervalSeconds 30 `
  -OutputDirectory artifacts/perf `
  -MaxRamMb 1200 `
  -MaxErrorLines 20
```

Expected outputs:
- `process-samples.csv`
- `summary.json`
- `summary.txt`

## Stress Scenarios (Control And Profiles)

### Automated

- `dotnet test FanControlPro.sln --filter FullyQualifiedName~ProfileSwitchStressTests`

### Manual

- Rapid profile switching for 5 minutes.
- Manual PWM changes every 2-3 seconds.
- Trigger `Full Speed` and return to previous profile.
- Validate no channel gets stuck and UI remains responsive.

## Baseline Targets

Track in `docs/qa/performance-baselines.csv`:
- idle CPU usage average (background),
- peak CPU usage during stress,
- average/peak working set memory,
- cold startup time,
- number of critical errors in logs.

## Crash Diagnostics Checklist

- Collect support bundle after each run.
- Extract critical log lines (`Error`, `Failsafe`, `Unhandled`).
- Classify issues P0/P1/P2.
- Link run ID to issue tracker ticket.

## Exit Criteria For 5.2

- At least one complete 24h run per target class (`FullControl` and `MonitoringOnly`).
- CPU background average <= 2% on validated reference targets.
- No critical unhandled exceptions during soak.
- Profile and settings persistence intact after stress/soak completion.
