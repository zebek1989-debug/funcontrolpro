# Getting Started (Public Release)

Last updated: 2026-03-15

## Install

1. Download latest `FanControl Pro` release from GitHub Releases.
2. Choose MSI installer (recommended) or portable ZIP.
3. Run installer and start the app.

## First-run setup

1. Complete onboarding.
2. Verify detected hardware and support mode.
3. Accept control-risk confirmation only if you plan to use write mode.

## Daily usage

- Monitor temperatures, RPM and system load from dashboard.
- Switch between profiles depending on workload.
- Use failsafe-friendly settings when testing new curves.

## Troubleshooting

- Export support bundle from diagnostics panel before opening an issue.
- Check known limitations: `docs/release/KNOWN_ISSUES.md`.
- Check compatibility status: `supported-hardware.md`.
